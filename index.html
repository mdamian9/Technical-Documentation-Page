New Repositories

Once you've created a new account on GitHub, you're ready to create your first project.
In order to do this, you can click on a button that is found in your account home page. It's a green button that says "New Repository."
You'll be able to create a new repository by clicking this button.

After you've clicked this button you'll now see the following on your screen. Follow the instructions provided.
You can name your project and provide a description. You can choose whether your repository is public or private.
If you select public, anyone can see your code, but you choose who can make edits to it. If you choose private, only you can see and edit your code.
If you'd like to add a README file to your project, you can check the box. Click on "create repository" and you're ready to go.
